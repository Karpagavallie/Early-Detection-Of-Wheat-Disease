# Early Detection of Wheat Diseases – Full Stack Web Application

Agriculture plays an important role in global food production, and wheat is one of the most widely cultivated crops. However, wheat plants are often affected by diseases such as rust, leaf blight, and powdery mildew, which can significantly reduce crop yield and quality. Early detection of these diseases is essential for effective crop management and prevention of large-scale losses.

This project presents a web-based system for the early detection of wheat diseases using image upload functionality. The application allows users to register, log in, and upload images of wheat leaves through an intuitive web interface. The backend processes the uploaded image and stores the analysis results in a MySQL database. The system then provides disease information, including possible symptoms, causes, and suggested treatments.

The project is developed using a full stack approach with HTML, CSS, and JavaScript for the frontend, Node.js with Express for the backend, and MySQL for database management. The system also maintains a history of uploaded reports, enabling users to track previous detections.

This application aims to assist farmers and agricultural researchers by providing a simple digital platform for identifying wheat diseases and managing crop health more effectively.
